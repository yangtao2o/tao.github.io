2019-06-12

Nodejs

定义

传输控制协议（TCP）是一个面向连接的协议，保证了两台计算机之间数据传输的可靠性和顺序。

Node HTTP 服务器是构建于 Node TCP 服务器之上的。

TCP 的特性：

  • 面向连接的通信和保证顺序的传递
  • 面向字节 - ASCII、Unicode
  • 可靠性
  • 流控制
  • 拥堵控制 - Qos

Socket 是对 TCP 协议的一种封装方式，Socket本身不是协议，而是一个编程接口。
